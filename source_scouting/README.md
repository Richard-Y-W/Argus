# Source scouting

Argus discovers questions from primary research, replication packages, official datasets, practitioner engineering sources, and public forums. Reddit/forums/blogs are unverified claims: useful for neglected operational problems, never evidence.

Store only URL, timestamp, community, title, aggregate metrics when available, and a research paraphrase. Do not store usernames or profile information. Every report records queries, date, selection bias, candidate ideas, and promotion decision.
